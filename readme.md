# 音乐控制器

- 突发奇想，我能不能躺在床上控制我电脑播放的音乐，可以飞歌的那种。

# 安装

1. 安装到酷Q应用目录
1. 修改插件配置
1. 重载插件或重启酷Q

# 使用

- 向机器人私聊窗口发送命令：
- 命令前缀 : `/` 或 `!`
- 命令不区分大小写

|  命令   | 功能描述     |
|---------|--------------|
| `NEXT`  | 播放下一首歌 |
| `PREV`  | 播放上一首歌 |
| `PAUSE` | 暂停/播放    |
| `VOL +` | 音量增加    |
| `VOL -` | 音量减小    |
